Глава I 


Введение

Этот раздел предназначен для ознакомления с системным администрированием. Это расскажет вам о важности использования сценариев для автоматизации ваших задач. Для этого вы откроете для себя технологию «докеров» и воспользуетесь ею для установки полноценного веб-сервера. Этот сервер будет запускать несколько сервисов: Wordpress, phpMyAdmin и базу данных SQL.
_______________________________

Глава II.

Общие инструкции


• Вы должны поместить все необходимые файлы для конфигурации вашего сервера в папку с именем srcs.
• Ваш файл Dockerfile должен находиться в корне вашего репозитория. Он построит ваш контейнер. Вы не можете использовать docker-compose.
• Все необходимые файлы для вашего сайта WordPress должны находиться в папке srcs.
______________________________________

Глава III.

• Вы должны настроить веб-сервер с Nginx только в одном контейнере докеров. ОС контейнера должна быть debian buster.
• Ваш веб-сервер должен иметь возможность запускать несколько служб одновременно. Сервисами будут сайт WordPress, phpMyAdmin и MySQL. Вам нужно будет убедиться, что ваша база данных SQL работает с WordPress и phpMyAdmin.
• Ваш сервер должен уметь использовать протокол SSL.
• Вам необходимо убедиться, что в зависимости от URL-адреса ваш сервер перенаправляет на правильный веб-сайт.
• Вам также необходимо убедиться, что ваш сервер работает с автоиндексом, который должен быть по возможности отключен.
